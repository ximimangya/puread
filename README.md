# PureAppData
一个使用 Web 管理的 Windows AppData 清理工具

![PureAppData](https://github.com/user-attachments/assets/b288dc5e-c9d7-428a-9d9f-d1759c1ee919)

# 使用
1. 下载源代码
2. 运行一次 install.bat
3. 复制快捷方式到开始菜单或桌面（可选）
4. 运行 start.vbs 即可自动打开浏览器开始清理
